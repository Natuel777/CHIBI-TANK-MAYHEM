\-------------- HOW TO MAKE PLAYER CONTROLLER FUNCTIONAL IN ANY SCENE ---------------------



The new scene created **MUST HAVE** the following prefabs configured as the instructions below indicate...



* **Main camera prefab** (The field 'Orbit Target' must be asigned with instance of the main player prefab set in scene, as well as the 'Input Reader' field.)
* **Tank prefab** (base configuration is set in the prefab, any changes must be added in the scene. But i endure adding new mechanics \& changes by code.)
* **Factory prefab** (contains all factories and pools, the prefab is already configured)
* **Game Manager prefab** (prefab is already configured)
* **Canvas prefab** (prefab is already configured)
